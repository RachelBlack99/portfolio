how do i

work

Research

Voor een project maak ik gebruik van de double diamond methode en human centered design. De researches zijn gebaseerd altijd op de target audience met een variatie in gebruik van onderzoeksmethoden (cmd-methodes). Zo kan ik mij aanpassen op elk project wanneer nodig. Op deze manier werken brengt mijn projecten tot beter resultaat.

Designen

Nadat er research is gehouden baseer ik mijn designkeuzes hierop. Mijn hoogste prioriteit is dat de doelgroep het design gebruiksvriendelijk vind en natuurlijk aanspreekt. Hiervoor maak ik gebruik van usertest voor prototypes. Het ligt natuurlijk aan het project hoe ik dit aanpak, maar ik maak meestal een keuze vanuit het cmd-methodes pack. Voor prototypes te testen gebruik ik userberry.

Development

Voordat ik begin met coderen maak ik altijd gebruik van een version control. Dit is om mijn code op een overzichtelijke manier te vertonen en dat het natuurlijk niet verloren kan gaan. Mijn codeer vaardigheden zijn voornamelijk in het webbased dus HTML, CSS en Javascript zijn mijn expertises.

Als laatste vind ik het heel belangrijk dat mijn code altijd gedocumenteerd is, zodat wanneer iemand anders er naar kijkt hij ook snapt wat ik heb gedaan.

Samenwerken

Mijn projecten zijn voor het meeste gedeelte altijd samenwerken. Ik kan goed samenwerken met andere mensen en vind het dan ook leuk om deze mensen te leren kennen. Zo kan ik namelijk dingen leren die ik nog niet wist. Op de fontys maken wij gebruik van SCRUM. Dit is een manier hoe een project verloopt onderling. Hiermee heb ik nu 2 jaar ervaring.
